# 🏥 Hospital Management System

A fully functional **Hospital Management System** built as a 2nd semester Database project at Air University. Developed using **C# and .NET WinForms** with **SQL Server** as the backend database.

---

## ✨ Features

- 🔐 **Login System** — Secure authentication for staff
- 🧑‍⚕️ **Patient Management** — Add, update, search patient records
- 📅 **Appointment Scheduling** — Book and manage appointments
- 💊 **Medical Records** — Store and retrieve patient medical history
- 🧾 **Billing System** — Generate and manage patient bills
- 🛏️ **Room Management** — Track room availability and assignments
- 👩‍⚕️ **Nurse Management** — Manage nursing staff records
- 🔍 **General Search** — Search across all modules

---

## 🛠️ Tech Stack

- **Language:** C#
- **Framework:** .NET WinForms
- **Database:** SQL Server
- **IDE:** Visual Studio

## ⚙️ How to Run

1. Clone the repository:
```bash
   git clone https://github.com/muhammadbilalkhalidmbk/hospital-management-system.git
```
2. Open `Hospital Management System.sln` in **Visual Studio**
3. Set up a **SQL Server** database and update the connection string in the code
4. Build the solution and run
